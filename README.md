# CompileIT Chat

A RAG-powered chatbot that answers questions about [compileit.com](https://compileit.com) in Swedish.

## Tech Stack

- **Backend:** Python, FastAPI, LangGraph
- **Frontend:** Next.js
- **Database:** PostgreSQL with pgvector
- **LLM:** OpenAI

## Project Structure

```
compileitchat/
├── backend/          # Python API
├── frontend/         # Next.js app
└── scripts/          # Scraping and indexing
```
